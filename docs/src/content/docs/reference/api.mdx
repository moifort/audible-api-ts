## Authentication

### `login(locale)`

Generate a login URL for the Audible PKCE OAuth flow.

```typescript
const { loginUrl, session, cookies } = await login('fr')
```

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| `locale` | `AudibleLocale` | The Audible marketplace to authenticate against |

**Returns:** `Promise<{ loginUrl: string, session: AuthSession, cookies: AudibleCookie[] }>`

---

### `register(authorizationCode, session)`

Complete device registration using the authorization code from the OAuth callback.

```typescript
const credentials = await register(code, session)
console.log(`Authenticated! Expires: ${credentials.expiresAt}`)
```

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| `authorizationCode` | `string` | The code from the OAuth callback URL |
| `session` | `AuthSession` | The session returned by `login()` |

**Returns:** `Promise<AudibleCredentials>`

**Throws:** `Error` if registration fails (invalid code, expired session, etc.)

---

### `refresh(credentials)`

Refresh an expired access token.

```typescript
const updated = await refresh(credentials)
console.log(`New expiration: ${updated.expiresAt}`)
```

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| `credentials` | `AudibleCredentials` | Current credentials with a valid `refreshToken` |

**Returns:** `Promise<AudibleCredentials>` ‚Äî New credentials with updated `accessToken` and `expiresAt`

**Throws:** `Error` if the refresh token is invalid

## Data Fetching

### `library(credentials)`

Fetch the user's entire Audible library with automatic pagination.

```typescript
const { items } = await library(credentials)

// Each item has full details: rating, categories, listening progress, etc.
items.map((book) => {
  const rating = book.rating?.overallDistribution
  console.log(`${book.title} ‚Äî ${rating?.averageRating?.toFixed(1)}/5`)
  if (book.listeningStatus?.finishedAt) {
    console.log(`  Finished on ${book.listeningStatus.finishedAt.toLocaleDateString()}`)
  }
})
```

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| `credentials` | `AudibleCredentials` | Valid credentials |

**Returns:** `Promise<{ items: AudibleItem[], credentials: AudibleCredentials }>`

Pagination is automatic ‚Äî all items are fetched (50 per page).

---

### `wishlist(credentials)`

Fetch the user's entire Audible wishlist with automatic pagination.

```typescript
const { items } = await wishlist(credentials)

items.map((book) =>
  console.log(`${book.title} ‚Äî ${book.durationMinutes} min ‚Äî ${book.publisher}`)
)
```

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| `credentials` | `AudibleCredentials` | Valid credentials |

**Returns:** `Promise<{ items: AudibleItem[], credentials: AudibleCredentials }>`

---

### `catalog(credentials, options)`

Search the Audible catalog by category with optional filters. Results are automatically sorted by number of ratings (desc) then average rating (desc).

```typescript
// Most voted Science Fiction (default ‚Äî fetches all pages, sorts by votes)
const { items } = await catalog(credentials, {
  category: 'science-fiction',
  numResults: 20,
})

// Or using raw category ID
const { items: items2 } = await catalog(credentials, {
  categoryId: '21229020031',
  numResults: 20,
})

items.map((book, i) => {
  const r = book.rating?.overallDistribution
  console.log(
    `${i + 1}. ${book.title} ‚Äî ${r?.averageRating?.toFixed(1)}/5 (${r?.numRatings} votes)`
  )
})
```

**Parameters:**
| Name | Type | Default | Description |
|------|------|---------|-------------|
| `credentials` | `AudibleCredentials` | ‚Äî | Valid credentials |
| `options.category` | `AudibleGenre` | ‚Äî | Genre name, resolved per locale |
| `options.categoryId` | `string` | ‚Äî | Raw category ID (alternative to `category`) |
| `options.sortBy` | `string` | `'MostVoted'` | Sort order (see below) |
| `options.maxPages` | `number` | `10` | Max pages for `MostVoted` (50 items/page) |
| `options.keywords` | `string` | ‚Äî | Search term filter |
| `options.author` | `string` | ‚Äî | Filter by author name |
| `options.narrator` | `string` | ‚Äî | Filter by narrator name |
| `options.numResults` | `number` | `50` | Number of results to return |
| `options.page` | `number` | `1` | Page number (for non-MostVoted sorts) |

**Sort options:** `'MostVoted'` (default ‚Äî fetches all pages, sorts by votes client-side), `'BestSellers'`, `'AvgRating'`, `'Relevance'`, `'ReleaseDate'`, `'-ReleaseDate'`, `'Title'`

**Returns:** `Promise<{ items: AudibleItem[], credentials: AudibleCredentials }>`

---

### `verify(credentials)`

Verify that credentials are valid by making a minimal API call.

```typescript
try {
  await verify(credentials)
  console.log('Connected')
} catch {
  console.log('Credentials expired ‚Äî call refresh()')
}
```

**Parameters:**
| Name | Type | Description |
|------|------|-------------|
| `credentials` | `AudibleCredentials` | Credentials to verify |

**Returns:** `Promise<void>`

**Throws:** `Error` if credentials are invalid or expired

## Configuration

### `AUDIBLE_LOCALES`

A record mapping each `AudibleLocale` to its marketplace configuration.

```typescript
import { AUDIBLE_LOCALES } from 'audible-api-ts'

AUDIBLE_LOCALES['fr']
// { domain: 'fr', marketplaceId: 'A2728XDNODOQ8T', countryCode: 'fr' }

AUDIBLE_LOCALES['com']
// { domain: 'com', marketplaceId: 'AF2M0KC94RCEA', countryCode: 'us' }
```

**Type:** `Record<AudibleLocale, LocaleConfig>`
