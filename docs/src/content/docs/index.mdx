## The TypeScript Audible Client

There is an excellent [Python Audible library](https://github.com/mkb79/Audible) by mkb79. But if you're building with TypeScript or Node.js, you've been on your own — until now.

**audible-api-ts** brings the same power to the TypeScript ecosystem: full PKCE authentication, signed API requests, paginated data fetching, and complete type definitions for every response.
